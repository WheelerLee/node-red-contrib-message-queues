Simple Message Queue

Input

typestring
  • add: Add a message to the queue
  • consume: Consume a message, indicating that the message has been processed, and automatically deliver the next message
payloadstring | number
The content of the message, either the newly added message or the consumed message
indexnumber
Optional, the position of the message in the queue, starting from 0. By default, it is added to the end of the queue

Output

payloadstring | number
The content of the message, indicating that the message has started being consumed
